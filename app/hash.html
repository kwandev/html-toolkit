Input

Drop a file here or click to choose — hashed locally, never uploaded

Output

SHA-1 / 256 / 384 / 512 use the browser's native crypto.subtle.digest (WebCrypto). MD5 is computed locally in JavaScript — it isn't a WebCrypto algorithm. Everything runs on your device; no text or file is ever uploaded.

Click any hash to copy it, or use Copy all for the whole list. Paste a checksum into verify to highlight the matching row. MD5 and SHA-1 are broken for security — use them only for checksums and legacy compatibility.
